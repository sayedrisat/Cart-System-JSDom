# LinkedIn Post

I recently built and published a Cart System using HTML, CSS, and vanilla JavaScript DOM.

This project helped me practice real frontend fundamentals without using any framework. The app allows users to create product cards, edit product details, delete products, and keep the data saved in the browser using localStorage.

What I practiced in this project:

- DOM selection with querySelector
- Event handling with addEventListener
- Dynamic UI rendering with innerHTML
- Form submission handling with preventDefault
- CRUD operations using an array of product objects
- localStorage data persistence with JSON.parse and JSON.stringify
- Product edit and delete functionality
- Popup form interaction using JavaScript

This project gave me a better understanding of how JavaScript connects user actions, data, and UI updates in a browser.
